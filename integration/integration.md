# Integration

Turbogit can plug itself with with some of your favourite tools whenever it makes sense.
For instance including a ticket id in the branch name.
If you think something is missing in your workflow with turbogit, do not hesitate to raise an issue on [b4nsy/turbogit](https://github.com/b4nst/turbogit/issues).

* [GitHub](/integration/github.md) - _Not available yet_
* [GitLab](/integration/gitlab.md) - Create new branches directly from GitLab issues
* [Jira](/integration/jira.md) - Create new branches directly from Jira issues
